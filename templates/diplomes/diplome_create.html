Documentation

Voici comment vous pouvez créer des diplômes en téléchargeant un fichier Excel :

  • Le fichier Excel doit avoir les colonnes suivantes :

    • - nom
    • - prenom
    • - date_de_naissance
    • - lieu_de_naissance
    • - sexe
    • - contact1
    • - contact2
    • - diplome
    • - niveau
    • - annee_academique
    • - date_soutenance (Formatter au format Text)
    • - session_soutenance
    • - cycle

  • Chaque ligne représente un diplôme avec les informations correspondantes.

  • Télécharger un modèle de fichier Excel ici.
